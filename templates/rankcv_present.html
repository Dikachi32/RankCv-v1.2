RankCV

AI-Powered CV Ranking System

Built with Flask by Dikachi

How It Works

01

User enters a job description and uploads CVs as PDF files. The system validates file types and enforces the user's plan limits (Free, Basic, Pro, or Premium) before accepting the upload.

02

PyPDF2 opens each PDF and extracts raw text from every page, stripping away images and formatting. Then scikit-learn takes over — its TfidfVectorizer converts the text into numerical vectors, and cosine_similarity scores each CV against the job description (0–100%).

Libraries: PyPDF2 + scikit-learn

TF-IDF & Scikit-Learn

TF

Term Frequency counts how often a word appears in a document. If "Python" appears many times in a CV, TF marks it as important.

"Python" appears 12 times → High TF score

IDF

Inverse Document Frequency down-weights words that appear in every document (like "the", "and", "is") so only unique, meaningful terms influence the ranking.

"the" appears in every CV → IDF score near zero

scikit-learn is the Python machine-learning library that powers RankCV.
It provides TfidfVectorizer to build the TF-IDF matrix and cosine_similarity to measure how close each CV vector is to the job description vector — producing the final 0–100% match score.
